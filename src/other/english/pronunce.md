### 音节

> 以元音为主体构成的发音单位，一般来说元音发音响亮，可以构成音节。

*   元音： a e i o u y w 这几个字母出现的时候说明有一个元音音节
    
*   辅音： b, c, d, f, g, h, j, k, l, m, n, p, q, r, s, t, v, w, x, y, z 注意（yw）也可以是辅音
    

```plaintext
// fast - 1个音节(Fast)
// order - 2个音节 (Or-der)
// tomorrow - 3个音节（To-mor-row）
```

## 单词原理总结

> 自己学习英语的一点总结

### 名词

名词表示人，事物，地点的名称。比如： dog.表示狗

### 形容词

形容词的作用就是修饰名词的，表示的是它所修饰的事物”看起来怎么样“如： a happy dog.

### 动词 

表示一个动作。如： run 这就是一个动词，表示奔跑

### 副词

用来修饰动词，它的作用是：修饰动作是”如何进行的“。如： he runs slowly.

用一句话来串联： the pround man specks loundly.
